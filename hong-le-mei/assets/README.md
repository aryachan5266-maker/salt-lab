# NΛCL Logo Assets

This folder contains transparent-background logo assets prepared from the provided brand image.

## Best files to use

- `svg/nacl-logo-white.svg` and `svg/nacl-logo-black.svg`: scalable wordmark for websites, print, presentations, and design tools.
- `svg/nacl-symbol-white.svg` and `svg/nacl-symbol-black.svg`: scalable standalone symbol / favicon source.
- `png/logo-white/` and `png/logo-black/`: transparent PNG wordmarks in 256, 512, 1024, 2048, and 4096 px widths.
- `png/symbol-white/` and `png/symbol-black/`: transparent PNG symbols in 16 to 1024 px sizes.
- `png/app-icon/`: black rounded-square app icon in common sizes.
- `png/favicon/favicon.ico`: multi-size favicon for websites.
- `extracted-from-original/`: raster versions cut from the original screenshot, useful if you want the exact original glow/texture.
- `preview/NACL-assets-preview.png`: quick visual preview sheet.

## Usage notes

Use SVG whenever possible. Use white versions on dark backgrounds and black versions on light backgrounds. PNG files are transparent unless they are app icons, which intentionally include the black rounded-square background.

## High-resolution lockups added

The `png/lockups-white`, `png/lockups-black`, and `png/4k-transparent` folders contain rebuilt high-resolution transparent logo lockups. These are not enlarged screenshots; the wordmark, Chinese text, English tagline, and glow line were redrawn for crisp 4K use.

Key rebuilt text:

- 盐 · 连接 · 系统 · Agent
- 连接一切 · 构建未来
- CONNECT EVERYTHING · BUILD THE FUTURE

Use `svg/lockups/nacl-primary-lockup-white.svg` or `svg/lockups/nacl-primary-lockup-black.svg` when you need scalable output.
