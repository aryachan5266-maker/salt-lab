# nacl-lab

盐究所 NACL-LAB 的公开电子简历与 AI 商业化作品中台。

当前版本重点是公开展示：

- AI 商业化定位与能力金字塔
- FleetFlow、Mango Z、试岗系统、mangoCRM、本地自动化案例
- 红了么、爆了么两条营销智能体产品线
- 已打码的系统证据碎片与动态展示层
- PC / 移动端响应式独立站

后续预留方向：

- 登录态
- 更多项目案例库
- 内部工具入口
- 协作型内容管理
- AI 产品化与销售闭环资料沉淀

## 品牌约定

品牌层级、Logo 使用、Demo 地址和数据纪律见 `BRAND_PROTOCOL.md`。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物会生成在 `dist/`，可用于 OSS、Netlify 或其他静态托管。

## 安全说明

公开目录只保留经过打码和二次处理的截图素材。原始系统截图、账号密码、真实金额、客户隐私、内部备注等不得提交到仓库。
