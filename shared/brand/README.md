# Shared Brand Assets

`shared/brand` is the source location for reusable NACL-LAB brand assets inside this monorepo.

- Source logo: `nacl-logo-white.svg`
- Public portal copy: `nacl-lab/public/assets/brand/nacl-logo-white.svg`
- Product apps may copy this asset into their own `public/` trees when they need a local deploy artifact.

Do not redraw the NACL logo or replace it with a temporary text mark.
